
# Housing Price Analysis (EDA)

## Project Overview

This project performs an Exploratory Data Analysis (EDA) on a housing dataset containing 545 residential properties. The objective is to identify the factors that influence house prices and provide insights that support informed home-buying decisions.

## Objectives

- Inspect and understand the dataset
- Clean and validate the data
- Explore relationships between house features and price
- Create visualizations to communicate findings
- Generate business insights and recommendations

## Tools Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Files Included

- `Housing_EDA.ipynb` – Complete data analysis notebook
- `Housing_Price_Analysis_Report.pdf` – Final project report
- `Housing.csv` – Dataset used for the analysis

## Key Findings

- Larger houses generally have higher selling prices.
- Houses in preferred areas command higher prices.
- Air conditioning and basements increase property value.
- Three-bedroom houses are the most common property type.
